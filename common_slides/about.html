Slide technicalities

  • Sound-enabled slides
    • Press "a" to start/pause the voice-over presentation
    • Press "esc" to get overview of slides
  • Navigate slides using arrow keys
    • "←" and "→" to move between major sections
    • "↑" and "↓" to dig deeper into current topic
  • Slide philosophy
    • Resonably standalone. I.e., works for both "live" traditional learning and "asynchronously" remote learning
    • Easily accessible and only relies on a simple web browser
    • Under version control on GitHub (you are most welcome to propose improvements and corrections)
    • You are always browsing the latest version
  • More info
    • Based on the reveal.js framework
    • Source code freely available at https://github.com/jakobhaervig/courses/
    • Contact me if you want to contribute to these slides
