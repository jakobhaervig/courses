Boundary condition types

The four boundary condition types

  • We have to specify boundary conditions on each non-internal node. Four types exist:

    • 1. Constant temperature

        • $T_0 = c$

    • 2. Constant heat flux

        • $\dot{Q}_\text{left,surface}=kA\frac{\text{d}T}{\text{d}x}=c$

    • 3. Convection

        • $\dot{Q}_\text{left,surface}=hA\left(T_\infty-T_0 \right)=c$

    • 4. Radiation

        • $\dot{Q}_\text{left,surface}=\epsilon \sigma A \left(T_\text{surr}^4-T_0^4\right)=c$

Incorporating the different boundary conditions

  • For BC type 1 (constant temperature), we simply fix the temperature
  • For BC type 2-4, we set up an energy balance on the volume element: $$ \small \begin{aligned} \small \sum_\text{all sides} \dot{Q} + \dot{G}_\text{element} = 0, \end{aligned} $$
    • which, in the case of a 1D plane wall, expands to

      • $$\small \dot{Q}_\text{left,surface} + kA \frac{T_1-T_0}{\Delta x}+\dot{g}_0\left(\frac{\Delta x}{2}A\right)=0.$$

  • Expression for $\dot{Q}_\text{left,surface}$ depends on BC type
